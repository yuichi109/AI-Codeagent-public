## AI Code Agent — スキルシステムの使い方

### スキルとは？
よく使う作業手順をエージェントに覚えさせたもの。
`/スキル名` と入力するだけでその手順を自律実行してくれる。

### 使い方
| 操作 | 入力例 |
|---|---|
| スキルを呼び出す | `/commit` |
| スキル候補を表示 | 入力欄で `/` を打つ（↑↓で選択、Tab/Enterで補完） |
| スキル一覧を見る | 「スキル一覧を見せて」 |
| スキルを新規作成 | 「今やった手順を `/スキル名` というスキルとして覚えて」 |
| スキルを修正 | 「`/commit` スキルを修正して、〇〇するように変えて」 |
| スキルを削除 | 「`/commit` スキルを削除して」 |

### 登録済みスキルの確認
「スキル一覧を見せて」と入力すると、登録済みスキルの名前と説明が表示される。

### スキルファイルの場所
`~/AI-Codeagent/skills/スキル名/SKILL.md`
追加・編集しても再起動不要で即反映される。
