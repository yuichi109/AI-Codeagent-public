## 使い方

```
/memory
/memory TEST1
```

引数でプロジェクト名を指定した場合はそのプロジェクトの MEMORY.md に保存する。
省略した場合は会話の文脈からプロジェクト名を推定する。

## 手順

1. 会話履歴からプロジェクト名を特定する（引数 > 文脈推定 の優先順）
2. 以下の形式で MEMORY.md の内容を作成する：

```
## 作業メモリ
最終更新: YYYY-MM-DD

### 現在地
（今どの作業フェーズにいるか・直前に何をしたか）

### 完了済み
- （完了したタスクを箇条書き）

### 次のタスク
- （次にやるべきことを優先順に箇条書き）

### 注意事項
（引き継ぎが必要な判断・ハマりポイント等があれば記載）
```

3. `write_file(path="<プロジェクト名>/MEMORY.md", content=<作成した内容>)` で保存する
4. 成功したら「✅ <プロジェクト名>/MEMORY.md を保存しました」とだけ返す

## 厳守事項

- 保存内容の解説・読み上げをしない
- 余計な作業（git add 等）をしない
- プロジェクト名が特定できない場合のみ確認する
